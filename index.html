About Me

MOON MAN

Moon man is a fully functional CRUD (create, read, update, delete) app, with authentication and authorization capabilities. This group project allows one to browse and create parts for a spaceship, then add them to a separate list of parts, where they can be removed.



REACT is used for the front-end, and Mongoose and MongoDB for the back end

WeatherWhich

WeatherWhich is an app for getting the weather in any major city in the world. Simply type in the name of the city, and see the current weather conditions there. Want a 3 day forecast? press the button that reads "forecast" and it will appear, press it again to make it dissappear.



This project is made utilizing HTML5, CSS3, vanilla javascript, and an API call to openWeathermap

Movie Massacre

Movie massacre is a movie blog with authentication and authorization capabilities. As a non-registered user, you can see the movies and posts, but cannot manipulate posts in any way. Once registered, the site remembers who you are, and remembers both the movies you have chosen to post about and the posts themselves. posts may also be deleted.

movie massacre is rendered in the front end usign REACT, with REACT ROUTER, and the database and routes on the back-end were set up using RUBY on RAILS. Bcrypt was used for authentication / authentication.
